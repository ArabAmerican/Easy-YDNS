Getting Started

YDNS is a free service that offers DNS hosting, including dynamic DNS.

If you're new to YDNS and like to sign up, click here. In case you already got an account, you can login right here.


Feature Overview

  • Dynamic DNS
  • Full IPv6 support, as well as support for IDN-based domains
  • Domain hosting (DNS only)
  • API for updating records
  • Community-based project

Contributing

YDNS started as a one-man project to provide dynamic DNS services and evolved into a mid-size project that offers simple DNS hosting services. We're currently looking for people who'd like to contribute to the source code of our software, as well as documentation contributions and supporters. If you like to contribute in any way you can contact us via email at support@ydns.io.
