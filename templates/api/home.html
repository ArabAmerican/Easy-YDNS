YDNS API

Welcome to the YDNS API Documentation.

These APIs allow third-party programs to manipulate YDNS records, most notably for dynamic DNS purposes. We have two API versions available:

  • API v1 — The legacy API, which is compatible with most devices and updaters out there.
  • REST API v2 — The new, JSON-based REST API.

For new implementations, please use the REST API v2 whenever possible.
